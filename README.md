# AMR Platform 1.0 重整版

本版針對兩個核心問題重新整理：

1. Portal 圖片上傳重做
   - 每次只選一張圖片
   - 既有 Portal／既有專區內容，上傳成功後立即寫入 Firebase
   - 不必再依賴第二次儲存才能看到圖片
   - 新增中的 Portal／內容會先保留草稿，完成標題後再儲存
   - 每個 Portal 與每篇內容最多 6 張內頁圖片

2. 人超盃管理修正
   - 管理中心的人超盃按鈕可正常開啟
   - 人超盃只選人員，不選隊伍
   - 可新增、修改、刪除、復原
   - 可勾選是否已達成
   - 已移除「統一達成門檻 100 分」欄位與顯示

更新方式：將 index.html 覆蓋 GitHub 專案根目錄原本的 index.html，再 Commit changes。
