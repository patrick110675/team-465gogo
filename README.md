# AMR Platform 1.0 正式版

上傳方式：
1. 解壓縮本 ZIP。
2. 將 `index.html` 與 `README.md` 上傳覆蓋到 GitHub 專案 `team-465gogo`。
3. Commit changes。
4. 開啟 https://patrick110675.github.io/team-465gogo/

本版功能：
- Firebase Firestore 連線
- 首頁 Dashboard：團隊 TOP3、個人 TOP3、公告、最新積分
- 積分：個人積分、團隊積分、隊伍共用選擇
- 排行榜：積分排行、人超盃、例會出席、歷史紀錄可修改/刪除
- QR：例會、夢想起飛、夜間 COP、軟性活動簽到
- Portal：固定版型，圖文活動 / PDF / 外部連結 / YouTube
- 管理中心：首頁設定、公告、隊伍、人員、Portal、主題中心
- 10 種主題

注意：
- 圖片可先用網址或小圖上傳轉成內嵌資料。正式大量圖片/PDF 之後建議接 Firebase Storage。
