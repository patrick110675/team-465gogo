# AMR 人力特攻隊 V5 正式版

功能：
- 深色專業 Dashboard
- 首頁冠軍 / MVP / 排行榜 / 最新紀錄
- 新增積分
- 團隊排行
- 個人排行
- 歷史紀錄
- 活動 QR Code 簽到：需要活動時才建立，不含簽退，同人同活動只能簽一次
- 設定管理：隊伍、成員、活動、系統設定
- 積分設定：項目固定，只能修改分數
- 匯出 CSV
- 晨會模式

Netlify 設定：
- Build command 留空
- Publish directory 留空

## V5.1 出席功能
- 每週三 AMR 可一鍵建立活動 QR
- 預設開始 09:00、寬限 5 分鐘
- 出席狀態：已到、遲到、請假、未到
- 每月請假 1 次，超過會提醒
- QR 簽到不含簽退
- 活動簽到名單可即時查看

## V5.2 QR 活動積分
- AMR 例會：只做出席管理，不加分。
- 夜創 / 夢想起飛 / 軟性活動：掃 QR 後自動依「積分設定」加個人與團隊分。
- 掃 QR 後可輸入新朋友人數，系統依原本分數自動計算。
- 新朋友分數同樣讀取原本積分設定，不另外設定第二套分數。
