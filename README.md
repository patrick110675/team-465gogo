# AMR Platform 1.0 完整整合版

本版整合：

- 人超盃改為純個人分數，不選隊伍。
- 管理中心新增「🏅 人超盃」。
- 可新增、修改、刪除、復原人超盃個人分數。
- 人超盃達成門檻統一設定一次。
- 排行榜的人超盃分數獨立顯示，不影響一般個人／團隊積分。
- Portal 與專區內容的內頁圖片改為一次上傳 1 張，最多 6 張。
- 每上傳 1 張會立即加入預覽，原本輸入的標題、貼文、時間、地點、網址不會消失。
- 圖片上傳到 Firebase Storage，FireStore 只儲存圖片網址。

## 更新方式

1. 解壓縮 ZIP。
2. 將 `index.html` 覆蓋 GitHub 專案根目錄原本的 `index.html`。
3. Commit changes。
4. 等 GitHub Pages 更新後重新整理網站；手機可用無痕頁面確認新版。

## 人超盃使用方式

管理 → 🏅 人超盃 → 選擇人員 → 輸入分數 → 儲存。

## Portal 單張上傳方式

管理 → Portal → 編輯專區／專區內容 → 每次選擇 1 張圖片 → 等上傳完成 → 重複直到最多 6 張 → 按儲存。
