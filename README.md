# AMR Team V7.5 Official Range Fix

本版已依照需求改好：
- 週次改成日期區間，例如 7/6–7/12
- 首頁顯示目前日期區間與本週冠軍
- 統計中心可選日期區間，看團隊排行、個人排行、積分總表、流水帳
- 積分新增時選日期區間，不再填第幾週
- QR 活動綁定日期區間
- 簽到管理顯示日期區間
- 歷史紀錄可依日期區間、隊伍、關鍵字篩選
- 管理中心可新增/刪除日期區間

## V7.5 Cycle Auto Update
- 管理中心新增「競賽週期自動產生」。
- 輸入競賽開始日與結束日後，自動產生每 7 天一個日期區間。
- 舊的積分、簽到、排行榜紀錄會保留。
- 系統會依照今天日期自動抓目前日期區間。


## V7.5 Auto 8 Weeks Update
- 自動維持至少 8 個日期區間。
- 規則：前 2 週 + 目前週 + 後 5 週。
- 每週會自動往後更新，不需要手動新增。
- 舊的積分、簽到、排行榜紀錄仍會保留，可在歷史紀錄與統計中心查詢。
