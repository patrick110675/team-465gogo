# AMR Platform 1.0 完整整合版 v3

本版把六個 Portal 全部改成「專區＋多篇內容」模式。

## Portal 使用方式
1. 進入「管理 → Portal」。
2. 先從左側選擇一個 Portal 並按「編輯」。
3. 下方會出現「專區內容」。
4. 可新增多篇：圖文活動、圖文內容、PDF、外部連結、YouTube。
5. 每篇可獨立放封面、最多 6 張內頁圖片、貼文、時間、地點、報名連結、PDF 與影片。
6. 每篇可修改、刪除、排序及顯示／隱藏。

原本的 Portal 六張封面與單頁資料會保留；新增內容後，點進 Portal 會先看到內容列表，再點入詳細頁。
